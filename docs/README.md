## Document Standards

- All new documentation should be placed in the appropriate category folder
- Use clear, descriptive filenames in SCREAMING_SNAKE_CASE.md format
- Include a summary/purpose section at the top of each document
- Keep the main README.md focused on getting started
- Update this index when adding new documentation
